# Samplaí

Seo áit do samplaí `lth` gur féidir a úsáid mar bhunchloch de do leathanaigh féin. Is féidir leathanach thar a bheith simplí a ghineadh go héasca:

```
lth {
	eolas {
		ainm { Mo Leathnach }
	}
	corp {
		croí {
			teideal {
				uimhir { 1 }
				g { Cur Teideal an Leathanaigh Anseo: }
			}
			alt {
				g {
					Scríobh alt iomlán anseo
				}
			}
			alt {
				g {
					Scríobh dara alt anseo
				}
			}
			teideal {
				uimhir { 3 }
				g { Cur fo-theideal anseo:  }
			}
			alt {
				g {
					Scríobh tríú alt anseo
				}
			}
		}
	}
}
```
